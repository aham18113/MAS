# Gun Detection Floor Simulation

This project is a Python simulation built with the Mesa framework.  
It models students, a shooter, and police agents moving on a 2D grid (a custom floor map) and reports casualty and evacuation statistics after each run.

---

## Requirements

- Python 3.x
- [mesa](https://mesa.readthedocs.io/)
- [matplotlib](https://matplotlib.org/)

Install the required packages with:

```bash
pip install mesa matplotlib
```
